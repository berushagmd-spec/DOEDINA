# 🟥DERD GENERATOR 95🟥

Рофл-бот с тремя отдельными генераторами персонажей. Каждый раз он создаёт новое
имя, выбирает случайный 24-битный HEX-цвет и перекрашивает соответствующую
картинку, сохраняя чёрные, белые и серые детали.

## Команды

- `/genderd` — случайный Дерд
- `/genrom` — случайный Ромикс Татар
- `/gennel` — случайный Дима Нелис
- `/help` — показать подсказку

Команды автоматически появляются в меню Telegram и работают в личных чатах,
обычных группах, супергруппах и темах. В группе также поддерживается форма
`/genrom@ИмяБота`.

## Правила генерации

### Дерд

Каждое имя обязательно заканчивается на `-ерд`, `-рд` или `-ед`. Процедурное
пространство содержит примерно 202 миллиарда сырых сочетаний.

### Ромикс Татар

Первая часть всегда заканчивается на `-икс`: например, «Ромикс», «Громикс» или
«Квадрикс». Вместо «Татар» выбирается случайная национальность, народ или этнос
из большой встроенной базы. Вместе это даёт более 4,5 миллиарда сочетаний.

### Дима Нелис

Имя выбирается из международной базы, причём славянские имена выпадают с
вероятностью 82%. Вторая часть всегда заканчивается на `-ис`. Пространство
сочетаний превышает 3 миллиарда.

## Быстрый запуск

1. Создай бота через `@BotFather` и скопируй токен
2. Переименуй `.env.example` в `.env`
3. Вставь токен в строку `BOT_TOKEN=...`
4. Установи зависимости и запусти:

```bash
python -m pip install -r requirements.txt
python bot.py
```

Токен нельзя выкладывать на GitHub. Файл `.env` уже добавлен в `.gitignore`.
На хостинге создай секретную переменную окружения `BOT_TOKEN`, а командой запуска
укажи `python bot.py`. Для worker-хостингов также приложен `Procfile`.

## Добавление в группу

Добавь бота как обычного участника. Администраторские права ему не нужны. Команды
видны при включённом Privacy Mode. Если в группе несколько ботов, Telegram может
добавить к команде имя бота — этот вариант тоже обрабатывается.

## Проверка без Telegram

```bash
python self_test.py
```

Тесты проверяют обязательные окончания всех трёх генераторов, базы имён и
этносов, новые команды, меню Telegram и корректность HEX-перекраски изображений.

## Файлы

- `bot.py` — Telegram Bot API, команды, меню, группы и long polling
- `name_generator.py` — генератор Дердов
- `character_generators.py` — генераторы Ромикса-Татара и Димы-Нелиса
- `image_generator.py` — HEX-перекраска трёх персонажей
- `cube_template.png` — шаблон Дерда
- `romix_template.png` — шаблон Ромикса Татара
- `nelis_template.png` — шаблон Димы Нелиса
- `self_test.py` — автономная проверка
- `.env.example` — пример настроек
- `requirements.txt` — единственная внешняя библиотека
