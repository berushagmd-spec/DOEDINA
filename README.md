# 🟥DERD GENERATOR 95🟥

Telegram-бот, который по команде `/generate` создаёт случайного Дерда:

- генерирует имя в духе «Дерд», «Доед», «Деед» и миллионов других вариантов
- каждое имя обязательно оканчивается на `-ерд`, `-рд` или `-ед`
- выбирает полностью случайный 24-битный HEX-цвет
- перекрашивает красную часть исходного куба, не трогая чёрные и белые детали
- отправляет готовую PNG-картинку с именем и HEX
- работает в личных чатах, обычных группах, супергруппах и темах
- сам добавляет `/generate` в меню команд

Команды `/generate`, `/GENERATE` и `/generate@ИмяБота` поддерживаются. Telegram
разрешает показывать в меню только команды из строчных английских букв, поэтому
подсказка в меню выглядит как `/generate`.

## Насколько большая база

В `name_generator.py` есть большая подборка готовых имён и процедурная база из
начал, гласных ядер, связок и окончаний. В текущей версии это примерно
202 миллиарда сырых сочетаний. Отдельный фильтр отбрасывает слишком длинные,
сломанные и нежелательные варианты, поэтому не требуется хранить гигантский
текстовый файл.

Точное число для текущей версии можно увидеть командой:

```bash
python name_generator.py
```

## Быстрый запуск

1. Создай бота через `@BotFather` и скопируй токен
2. Переименуй `.env.example` в `.env`
3. Вставь токен в строку `BOT_TOKEN=...`
4. Установи зависимости и запусти:

```bash
python -m pip install -r requirements.txt
python bot.py
```

Токен нельзя выкладывать на GitHub. Файл `.env` уже добавлен в `.gitignore`.
На хостинге лучше создать секретную переменную окружения `BOT_TOKEN`, а командой
запуска указать `python bot.py`. Для worker-хостингов также приложен `Procfile`.

## Добавление в группу

Добавь бота как обычного участника. Администраторские права ему не нужны. Команды
видны даже при включённом Privacy Mode, поэтому достаточно написать `/generate`.
Если в группе несколько ботов, Telegram может отправить
`/generate@ИмяТвоегоБота` — этот вариант тоже обработается.

## Проверка без Telegram

```bash
python self_test.py
```

Тест проверяет объём пространства имён, 10 000 генераций, регистр команд и то,
что перекраска сохраняет чёрные/белые части исходного изображения.

## Файлы

- `bot.py` — Telegram Bot API, меню, группы и long polling
- `name_generator.py` — огромная процедурная база имён
- `image_generator.py` — перекраска куба
- `cube_template.png` — исходный куб
- `self_test.py` — автономная проверка
- `.env.example` — пример настроек
- `requirements.txt` — единственная внешняя библиотека
